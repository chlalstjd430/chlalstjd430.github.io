문제 풀이

다이나믹 프로그래밍(Dynamic Programing) 문제를 풀 때에는 항상 문제의 조건과 하나의 값을 가질 수 있는 경우이다.

현재 문제에서의 조건은 다음과 같다.


1. 스티커는 2N의 크기이다.

2. 뗀 스티커의 왼쪽,위,오른쪽,아래의 스티커는 사용 할 수 없다.


따라서 왼쪽부터 스티커를 뗀다고 하면, 최종적으로 마지막 (0,N),(1,N)의 스티커 중 하나는 뗴어져야한다. 이를 가지고 조건을 만든다면.

1) 마지막 스티커가 첫번째 줄인 경우 - (0,N)

(1) 이전 스티커가 현재 스티커 기준으로 왼쪽 대각선 아래인 경우 - (1,N-1)

현재 스티커까지의 합 = 현재 스티커 + 왼쪽 대각선 아래 스티커까지의 합

dp([0][N]) = [0][N] + dp([1][N-1]) 

(2) 이전 스티커가 현재 스티커 기준으로 왼쪽에서 두번째인 경우 - (0, N-2)

현재 스티커까지의 합 = 현재 스티커 + 왼쪽 두번째 스티커까지의 합

dp([0][N]) = [0][N] + dp([0][N-2])

(3) 이전 스티커가 현재 스티커 기준으로 왼쪽에서 두번째, 대각선 아래인 경우 - (1, N-2)

현재 스티커까지의 합 = 현재 스티커 + 왼쪽 두번째,대각선 아래 스티커까지의 합

dp([0][N]) = [0][N] + dp([1][N-2])


- dp([0][N]) = (1),(2),(3) 중에 가장 큰 값


2) 마지막 스티커가 두번째 줄인 경우 - (1,N)

(1) 이전 스티커가 현재 스티커 기준으로 왼쪽 대각선 위인 경우 - (0,N-1)

현재 스티커까지의 합 = 현재 스티커 + 왼쪽 대각선 위 스티커까지의 합

dp([1][N]) = [1][N] + dp([0][N-1]) 

(2) 이전 스티커가 현재 스티커 기준으로 왼쪽에서 두번째인 경우 - (1, N-2)

현재 스티커까지의 합 = 현재 스티커 + 왼쪽 두번째 스티커까지의 합

dp([1][N]) = [1][N] + dp([1][N-2])

(3) 이전 스티커가 현재 스티커 기준으로 왼쪽에서 두번째, 대각선 위인 경우 - (1, N-2)

현재 스티커까지의 합 = 현재 스티커 + 왼쪽 두번째,대각선 위 스티커까지의 합

dp([1][N]) = [1][N] + dp([0][N-2])


- dp([1][N]) = (1),(2),(3) 중에 가장 큰 값


최종적으로 dp[0][N]과 dp[1][N] 중에 큰 값이 답이 될 수 있다. 


이를 통해 스티커와 DP 배열을 선언한다.

#include <iostream>
#include <algorithm>
 
using namespace std;
 
int sticker[2][100002];//스티커를 저장할 배열
int dp[2][100002];//dp 전용 배열
 
int main() {
    int tc,n;
    cin >> tc;//테스트 케이스 입력
 
    for (int i = 0; i < tc; i++) {//테스트 케이스 만큼 반복
        cin >> n;//스티커 크기 입력
 
        for (int j = 0; j < 2; j++) {//스티커 값 입력
            for (int k = 1; k <= n; k++) {
                cin >> sticker[j][k];
            }
        }
 
        //dp 첫번째 값 입력
        dp[0][1] = sticker[0][1];
        dp[1][1] = sticker[1][1];
        //dp 구하기
        for (int j = 2; j <= n; j++) {
            dp[0][j] = sticker[0][j] + max(dp[1][j - 1], max(dp[0][j - 2], dp[1][j - 2]));
            dp[1][j] = sticker[1][j] + max(dp[0][j - 1], max(dp[0][j - 2], dp[1][j - 2]));
        }
 
        //최종 값 출력
        cout << max(dp[0][n], dp[1][n]) << endl;
 
        //스티커와 DP 배열 초기화
        for (int j = 0; j <= n; j++) {
            dp[0][j] = sticker[0][j] = 0;
            dp[1][j] = sticker[1][j] = 0;
        }
    }    
}
