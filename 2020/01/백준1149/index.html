문제 풀이

해당 문제에서 요구하는 것은 하나의 집에 대헤서 이웃하는 집(i-1,i+1)과 다른 페인트를 색칠하게 하는 것이다. 그리하여 모든 집을 칠하는 최소 페인트 비용을 구하면된다.

우선 하나의 집에 대헤서 각각의 페인트 비용을 저장하기 위해 다음과 같은 전역 변수를 선언해주었다.

int cost[1001][3];

실제로 문제의 예제를 가지고 테스트하면

1. 첫번째

R : 26[R]

G : 40[G]

B : 83[B]

 

2. 두번째 

R : 49[R] + 40[첫번째의 G]

G : 60[G] + 26[첫번째의 R]

B : 57[B] + 26[첫번째의 R]

 

4. 세번째 

R : 13[R] + ( 57 + 26[첫번째의 R])[두번째의 B] = 96

G: 89[G]…

B : 99[B]…

최종적으로 나온 값의 가장 작은 값이 예체의 출력값과 같은 것을 확인 할 수 있다.
